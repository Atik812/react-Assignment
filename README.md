## 🚀 Getting Started (Local Development)

To run this project locally:

1. **Clone the repo**
   ```bash
   git clone https://github.com/Atik812/react-Assignment.git
   cd react-Assignment
2. Install dependencies
   npm install
3. Run the dev server
   npm run dev
4. Open your browser at
   👉 http://localhost:5173
